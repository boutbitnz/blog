Getting Hands On

Apr 16th, 2015 12:00 am

WEEK 25 below more This week at the YWebCA, thing were wonderful. I can honestly say that so far this has to be one of my favorite weeks. I’ll tell you why..

Working with this Eloquent Javascript material is challenging, it’s calculating, criticle at times because Im learning so much so fast that I wasn’t really sure of myself. Was uncertain about my capabilities and performance. I was concerned about forgetting the code or my abilities in using what I’ve learned in making things do what I wanted them to do..

This week, I was able to overcome those obstacles in my thinking. This week not only did I really start to paste together an actual functioning web page, but I am almost complete with it. I am having so much fun building my page exactly the way I want it (structured wise) and styling my page that I framed out. Whats even more exciting is that i realized then that ‘I can do this, this is definitely the class for me.’ My true ‘Awe Hah’ moment.

The first thing I did when starting my project was created my directory, folder, and touch files that I would Use to store all my project material in. I did all that through the terminal. That was awesome to do on my own. once I completed that I went straight to work on everything else.

I put that page together with no problems. First I build my html. In doing so I took my time in making sure that everything stayed organized and in order I also payed a lot of attention to my cascading so if i was to run into any problems it wouldn’t be hard to navigate through the page. After completing what I could there I started to dress her up with a little css, Which I also payed close attention to keeping organized and in a good readable order. Right now I am working on getting the page to function the way I want with the javascript and jQuery I’ve learned so far. So every things going pretty good right mow. I’m feeling quite confident at the moment.

Another interesting thing I learned this week is ‘ Conway’s Game of Life ’ and 'Cellular automaton’ An initial state is selected by assigning a state for each cell. Thats crazy! Watching those cell interact is amazing and being able to control the flow (somewhat) makes it even more fun.

I am totally looking forward this coming week. ready to broaden horizons.
